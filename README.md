CLIENT/SERVER SỬ DỤNG GIAO THỨC TCP/IP


App sẽ gồm các chức năng


  1.Chức năng đăng nhập: chức năng này cho phép người dùng đăng nhập vào
giao diện chat.

  2. Chức năng gửi tin nhắn : chức năng này cho người dùng chat với nhau bằng
cách gửi tin nhắn qua lại với nhau.

  4. Chức năng truyền file: Truyền file qua lại giữa các client với nhau. Lưu file
vào hệ thống.

  5. Chức năng gọi điện : chức năng này cho phép người dùng gọi điện với nhau
   
  6. Chức năng chat nhóm: chức năng này cho phép người dùng có thể chat 
được nhiều người với nhau

APP chạy trên desktop

Ngôn ngữ sử dụng Java
